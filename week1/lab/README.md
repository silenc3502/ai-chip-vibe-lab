# week1/lab

학생이 1주차 동안 만드는 코드가 들어가는 폴더입니다.

**시작 시점에는 비어있습니다.** 회차 진행에 따라 본인이 바이브 코딩으로 파일을 추가합니다.

예상 파일 (회차별):

| 회차 | 파일 | 비고 |
| --- | --- | --- |
| 1-1 | `01_loop_vs_numpy.py` | for-loop vs NumPy 행렬곱 |
| 1-2 | `02_cache_effect.py` | 캐시 효과 관찰 |
| 1-3 | `03_simd_intro.py` | 벡터화 비교 |
| 1-4 | `04_matmul_baseline.py` | NumPy MatMul + GPU 모델 비교 |
| 1-5 | `05_cpu_gpu_npu.py` | 세 구조 비교 정리 |

> 파일명은 권장이며, 본인 흐름에 맞게 조정해도 됩니다. 단 회차 번호 prefix는 유지하세요 — 채점 시 시간순 흐름을 봅니다.

참조 솔루션은 회차 종료 후 `../reference/`에 공개됩니다.
